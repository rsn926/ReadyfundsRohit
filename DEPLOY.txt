🎯 DEPLOYMENT STEPS (5 MINUTES):
================================

STEP 1: Extract the ZIP
------------------------
Unzip readyfunds_SIMPLE.zip to a folder

STEP 2: Upload to GitHub
-------------------------
A. Create new repository on GitHub.com
   - Name it: "readyfunds-analytics" (or any name)
   - Public or Private (your choice)

B. Upload ALL 6 files:
   ✓ app.py
   ✓ requirements.txt
   ✓ .gitignore
   ✓ README.md
   ✓ msme_refined_dataset.xlsx
   ✓ .streamlit folder (with config.toml inside)

   METHOD 1 - GitHub Web Interface:
   - Click "uploading an existing file"
   - Drag all files
   - Commit

   METHOD 2 - Git Command Line:
   cd readyfunds_simple
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/readyfunds-analytics.git
   git push -u origin main

STEP 3: Deploy on Streamlit Cloud
----------------------------------
1. Go to: https://share.streamlit.io

2. Sign in with GitHub

3. Click "New app"

4. Select your repository:
   - Repository: YOUR_USERNAME/readyfunds-analytics
   - Branch: main
   - Main file path: app.py

5. Click "Deploy"

6. Wait 2-3 minutes

7. ✅ DONE! Your app is live!

   ERROR: "No module named 'xgboost'"
   FIX: Wait - dependencies are installing
   
   ERROR: "Memory limit exceeded"
